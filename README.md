# PWA progressive web app to track the status of startups
Uses google drive to store PDFs in folder "Startup Investor PDFs"
Uses google sheets to store startup information in "Startup-Status"
The email of agent is agent-scout@startup-status.iam.gserviceaccount.com

# Files
startups.json is the list of startups by name and a url to their website
pyscript.json (sets up the python environment with a list of libraries)
index.html (the dashboard)
manifest.json (the instructions for the app PWA, makes the add to home screen on iphone work)
